    # styled-components的简单用法

    Styled-components改变了简单的样式和组件的映射关系。将样式组件化（css in js）,
让我们能够模块化开发、参数定制，也能够更加灵活的定义组件的样式和复用样式。

    # 安装项目依赖

    npm install --save styled-components
    1

    # 基本用法

    定义样式组件和使用

        ##定义样式组件
    const Title = styled.h1`
                    font-size: 1.5em;
                    text-align: center;
                    color: palevioletred;
                    `;
    const Wrapper = styled.section`
                    padding: 4em;
                    background: papayawhip;
                    `;
    ##使用样式组件
    render(
            <Wrapper>
                <Title>
                Hello World, this is my first styled component!
                </Title>
            </Wrapper>
            );

    # 属性传递

    样式组件能够将props定义的标签属性，传递到对应的标签上

        ##定义样式组件
    const Input = styled.input`
                padding: 0.5em;
                margin: 0.5em;
                color: palevioletred;
                background: papayawhip;
                border: none;
                border-radius: 3px;
                `;
    ##使用样式组件并定义props标签属性
    render(
        <div>
            <Input placeholder="@mxstbr" type="text" />
            <Input value="@geelen" type="text" />
        </div>
        );

    # 基于props定制样式

    根据组件传递的props，可以内部使用插值法(函数)，响应样式定制

        ##定义样式组件
    const Button = styled.button`
        background: ${props => props.primary ? 'palevioletred' : 'white'};
        color: ${props => props.primary ? 'white' : 'palevioletred'};
        font-size: 1em;
        margin: 1em;
        padding: 0.25em 1em;
        border: 2px solid palevioletred;
        border-radius: 3px;
        `;
    ##使用样式组件
    render(
            <div>
                <Button>Normal</Button>
                <Button primary>Primary</Button>
            </div>
            );

    # 样式组件的继承

    通过extend方法，能够继承组件的样式，样式得到复用，提高开发效率

        ##定义样式组件
    const Button = styled.button`
                color: palevioletred;
                font-size: 1em;
                margin: 1em;
                padding: 0.25em 1em;
                border: 2px solid palevioletred;
                border-radius: 3px;
                `; 
    const TomatoButton = Button.extend`
                color: tomato;
                border-color: tomato;
                `;
    ##使用样式组件
    render(
            <div>
                <Button>Normal Button</Button>
                <TomatoButton>Tomato Button</TomatoButton>
            </div>
    );

    # 样式组件标签名的更换

    withComponent('tagName')可以将一个样式组件的标签名进行替换

        ##定义样式组件
    const Button = styled.button`
                display: inline-block;
                color: palevioletred;
                font-size: 1em;
                margin: 1em;
                padding: 0.25em 1em;
                border: 2px solid palevioletred;
                border-radius: 3px;
                `;
    const Link = Button.withComponent('a')
    const TomatoLink = Link.extend`
                color: tomato;
                border-color: tomato;
                `;  
    ##使用样式组件
    render(
            <div>
                <Button>Normal Button</Button>
                <Link>Normal Link</Link>
                <TomatoLink>Tomato Link</TomatoLink>
            </div>
            );

    # 扩展props

    通过attrs({config})可以给标签添加动态和静态的属性，扩展组件接收的props

        ##定义样式组件
    const Input = styled.input.attrs({
                    type: 'password',
                    margin: props => props.size || '1em',
                    padding: props => props.size || '1em'
                    })`
                    color: palevioletred;
                    font-size: 1em;
                    border: 2px solid palevioletred;
                    border-radius: 3px;
                    margin: ${props => props.margin};
                    padding: ${props => props.padding};
                    `;
    ##使用样式组件
    render(
            <div>
                <Input placeholder="A small text input" size="1em" />
                <br />
                <Input placeholder="A bigger text input" size="2em" />
            </div>
            );

    # animation

    通过keyframes可以生成唯一的名字，可以有效的避免命名冲突

        ##定义样式组件
const rotate360 = keyframes`
                from {
                    transform: rotate(0deg);
                }
                to {
                    transform: rotate(360deg);
                }
                `;
const Rotate = styled.div`
            display: inline-block;
            animation: ${rotate360} 2s linear infinite;
            padding: 2rem 1rem;
            font-size: 1.2rem;
            `;
    ##使用样式组件
    render(
        <Rotate>&lt; 💅 ">&gt;</Rotate>
    );
