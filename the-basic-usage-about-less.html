    # 变量的使用（varilables）

    使用关键字**@**+自定义标识符，它通常用来代替一个重复用到的样式属性值

            @header-bg:red;
        @header-height:100px;
        div{
            background-color: @header-bg;
            height: @header-height;
        }
    #编译后
        div {
            background-color: #ff0000;
            height: 100px;
        }

    # 混合的使用（mixins）

    混合的使用，是将定义好的类名直接整体移入到调用的地方，相当于在调用处，新建一个样式块。
这种方法的使用，在定义混合的时候没用加上（），所以在编译后，依旧会保留相关的类定义 

        #定义两个不带小括号混合
        .fl{ float: left }
        .fr{ float: right }
    #定义两个调用
        .left{ .fl() }
        .right{ .fr() }
    #编译后
        .fl { float: left }
        .fr { float: right }
        .left { float: left }
        .right { float: right }

    # 参数混合（parametric_mixins）

    参数混合的目的是：可以灵活的自定义样式值，它有点类似于变量加混合的结合用法

       #定义带小括号的参数混合，并在小括号中使用变量（同时可以设置参数值）
        .fl() { float: left }
        .fr() { float: right }
        .left {
                .fl();
                ##这里会使用默认参数
                .border-radius();
            }
        .right {
            .fr();
            ##这里会使用自定义参数
            .border-radius(15px)
        }
        .border-radius(@r: 50%) {
            border-radius: @r
        }
    #编译后
        .left {
            float: left;
            border-radius: 50%;
        }
        .right {
            float: right;
            border-radius: 15px;
        }

    # 参数逻辑条件混合(mixin_guards)

    它的使用是**#**开头，在加上逻辑混合的写法，最后加上when(变量判断)

        #定义一个参数逻辑混合
        #fontSize(@px) when (@px=small){
            font-size:12px;
        }
        #fontSize(@px) when (@px=middle){
            font-size: 24px;
            color: yellow;
        }
        #fontSize(@px) when (@px=big){
            font-size: 30px;
            color: blue
        }
    #调用
        .left {
        .fl();
        .border-radius();
            #fontSize(big)
        }
        .right {
            .fr();
            .border-radius(15px);
            #fontSize(middle)
        }
    #编译后
    .left {
        font-size: 30px;
        color: #0000ff;
        }
        .right {
        font-size: 24px;
        color: yellow;
        }

    # 嵌套规则（Nesting）

    嵌套规则可以快速的定义样式的层级关系，也能明显的反映出html的文本结构

       #第一种嵌套是使用空格嵌套，它反映的是html的层级关系
   #定义一个嵌套
    .parent{
        .son{
            color:red;
        }
        }
    #编译后
        .parent .son{
            color:red;
        }
    #第二种嵌套是使用&连接嵌套，它代表当前嵌套父级的前缀(引用父级名)
    #定义一个嵌套
        .parent{
            &._son{
            color:red;
            }
        }
    #编译后
        .parent_son{
            color:red;
        }
